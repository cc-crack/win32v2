RtlInitUnicodeString function

Initialises a UNICODE_STRING structure from a null-terminated Unicode string.

Syntax (C++)

void __stdcall RtlInitUnicodeString(_Out_ UNICODE_STRING* DestinationString, _In_opt_ const wchar_t* SourceString);

Parameters

DestinationString

Points to the UNICODE_STRING structure that will be initialised.

SourceString

Points to a null-terminated string of Unicode characters that will initialise the structure. This parameter can be NULL.

Return value

This function does not return a value.
