SRWLOCK structure

Syntax (C++)

struct SRWLOCK
{
      
  intptr_t Value;
};

Members

Value

This member is a bitfield. The bits have the following meanings:
  • Bit 0 - Indicates whether the SRW lock is locked.
  • Bits 1-3 - Reserved for internal use by the operating system.
  • Bits 4-15 - Shared acquisition recursion count. This can only be nonzero if the SRW lock is locked. If the SRW lock is locked in exclusive mode, this is 0.
  • Bits 16-31 - Reserved for internal use by the operating system.

Remarks

When initialising a lock, assign 0 to the lock, also SRWLOCK_INIT in the Win32 headers.
