UNICODE_STRING structure

Represents a Unicode string.

Syntax (C++)

struct UNICODE_STRING
{  unsigned short Length;
  unsigned short BufferLength;
  unsigned wchar_t* Data;
}

Members

Length

The length of the Unicode string, in bytes, excluding the null terminator.

BufferLength

The length of the buffer containing the Unicode string, in bytes. RtlInitUnicodeString initialises this member to the number of bytes in the Unicode string, excluding the null terminator.

Buffer

A pointer to the buffer containing the Unicode string.
